# v-else

The `v-else` conditionally renders an element or a template fragment as a function in case the `v-if` does not fulfil the condition.

## Example

```html
<div v-if="Math.random() > 0.5">
  Now you see me
</div>
<div v-else>
  Now you don't
</div>
```
